# 行为准则

SkiLens 采用 Contributor Covenant Code of Conduct 2.1 的精神。

## 我们的承诺

我们希望这个项目对所有参与者都是友好、专业、尊重的。不论年龄、身材、可见或不可见的残障、民族、性别特征、性别认同与表达、经验水平、教育背景、社会经济状态、国籍、外貌、种族、宗教、性取向或其他身份，所有人都应被尊重。

## 鼓励的行为

- 使用友好、包容的语言。
- 尊重不同观点和经验。
- 接受建设性反馈。
- 关注项目和社区的长期利益。
- 对其他参与者保持同理心。

## 不接受的行为

- 性化语言或图像。
- 挑衅、侮辱、人身攻击。
- 公开或私下骚扰。
- 未经允许发布他人隐私信息。
- 其他在专业环境中明显不合适的行为。

## 执行方式

维护者负责解释和执行本行为准则。维护者可以删除、编辑或拒绝不符合行为准则的评论、commit、代码、issue 和其他贡献。

如果需要报告行为准则问题，请优先使用 GitHub private vulnerability reporting。如果仓库尚未开启该功能，请创建一个标题为 `维护者联系请求` 的公开 issue，但不要写私人细节。维护者会提供私下沟通渠道。
